# Задание 

[Ссылка на условие](https://ai.hse.ru/hacks/psb24)
Задача классификации.
В рамках данного хакатона нам надо было создать модель для банка, которая помогала бы отелям-партнерам **определять отменит ли клиент бронирование номера**. Результаты технической части надо было внедрить в **новый банковский продукт**, помогающий студентам с организацией отдыха, а отелям с пониманием, как не понести убытки из-за отмененных броней.
